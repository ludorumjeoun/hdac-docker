# HDAC Friday Node Docker

## 빌드하고 노드 실행하기
```
git clone git@github.com:ludorumjeoun/hdac-docker.git
cd ./hdac-docker
./start friday
```

## cli 실행하기
```
./nodef friday ...
./clif friday ...
```

## 컨테이너에서 작업하기
```
./shell friday
```
